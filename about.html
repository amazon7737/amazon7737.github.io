    # About

    웹 애플리케이션 서비스를 만드는 것에 관심이 있습니다. 아이디어를 생각하는 것에 재미를 느끼고 사람들이 친근하게 사용하는 서비스를 만들고 싶습니다. 같이 개발하는 친구에게 어떤 도움이 될 수 있을까를 고민하며 API 서버를 만들고 Service 테스트 코드를 작업하며 데이터를 다루는 개발에 재미를 느꼈습니다. 터미널과 맥북은 저의 친구입니다.
